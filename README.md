# Landing Page Demo

A modern, interactive landing page demo built with Next.js and Tailwind CSS. Features a dark-themed design with glassmorphism effects, 3D floating shapes, and smooth animations.

<img width="1365" height="598" alt="lumia1" src="https://github.com/user-attachments/assets/e2a7300a-79c9-4bf6-aea2-c5a7180182ef" />


## Key Features

- Performance Optimized: Built on Next.js for fast load times and SEO.
- Premium Design:
  - Glassmorphism: Sleek, frosted-glass effects throughout the UI.
  - 3D Floating Shapes: Dynamic, interactive 3D-like objects that react to user interaction.
  - Spotlight Effect: An intelligent mouse-following spotlight that illuminates content.
- Interactive Components:
  - Hero Section: Engaging entrance with smooth entry animations.
  - Adaptive Navbar: Responsive navigation with glassmorphism.
  - Dynamic Pricing: Custom-styled pricing tables with hover effects.
  - Testimonial Slider: Clean and impactful feedback section.
- Fully Responsive: Optimized for all screen sizes, from mobile to ultra-wide displays.

## Tech Stack

- Framework: Next.js (App Router)
- Styling: TailwindCSS
- Animations: Framer Motion
- Icons: Lucide React
- Language: TypeScript

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd landing-page-demo
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open the application:
   Navigate to http://localhost:3000 to view the landing page.

## Customization

The project is built to be highly customizable. You can modify the global styles in `src/app/globals.css` or update component-specific logic in the `src/components/` directory.
